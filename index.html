Features

Everything You Need to
Manage Academics

A comprehensive suite of tools designed for departments to streamline their academic workflows.

Student Management

Add and manage student records with auto-generated login credentials. Track enrollments by course and roll number.

Faculty Management

Register faculty members with department info and auto-login creation. Assign subjects to faculty seamlessly.

Subject Assignment

Create subjects and assign them to faculty members. Track which professor teaches which course effortlessly.

Result Tracking

Record and manage student marks per subject and semester. Students can view their own results in a clean interface.

Assignment Submissions

Create assignments, set due dates, and receive file submissions. Supports PDF, ZIP, DOC, PPT formats up to 10 MB.

Role-Based Access

Three distinct user roles — Admin, Faculty, and Student — each with tailored dashboards and permission-controlled views.

User Roles

Three Roles, One System

Each role sees a tailored dashboard with relevant actions and data.

Admin

  • Full system dashboard with stats
  • Manage students & faculty
  • Create subjects & assign faculty
  • Record & view all results
  • Create & manage assignments
  • Approve student submissions

Faculty

  • Personalized subject dashboard
  • View pending submissions
  • Add students to the system
  • Record marks for students
  • Create & assign homework
  • Review & approve submissions

Student

  • Personal academic dashboard
  • View assignments (card layout)
  • Upload assignment submissions
  • View semester-wise results
  • Track submission statuses
  • See recent marks & grades

Get Started

Setup in Minutes

Follow these steps to get DAMS running on your local machine.

1

Clone the Repository

Download the project from GitHub into your XAMPP htdocs directory.

git clone https://github.com/Shripad-24/Depertmental--Academics-Management-System.git dams
2

Move to XAMPP Directory

Place the project folder inside C:\xampp\htdocs\dams\ so Apache can serve it.

3

Start XAMPP Services

Open XAMPP Control Panel and start Apache and MySQL services.

4

Import the Database

Open phpMyAdmin at http://localhost/phpmyadmin and import the sql_queries.sql file. This creates all tables and sample data.

5

Configure Database Credentials

Edit config/db.php if your MySQL credentials differ from the defaults (root / root).

6

Launch the Application

Open your browser and navigate to:

http://localhost/dams/
